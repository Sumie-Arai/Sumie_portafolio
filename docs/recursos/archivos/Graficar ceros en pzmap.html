1. ¿Qué hace pzmap?

La función pzmap(sys) grafica los polos (×) y los ceros (o) de un sistema dinámico.
Sirve para analizar estabilidad y la respuesta en frecuencia/tiempo de tu sistema.

  • Polos: determinan la dinámica (oscilaciones, rapidez, estabilidad).

  • Ceros: afectan la forma de la respuesta, por ejemplo introducen sobrepaso, anulan modos del sistema o producen cancelaciones.

2. Ejemplo sencillo de práctica de Control Continuo

Supongamos una planta típica de primer orden con retardo en la entrada:

G(s)=10s2+3s+2G(s) = \frac{10}{s^2 + 3s + 2}G(s)=s2+3s+210​

Aquí:

  • Numerador (num) = [10]

  • Denominador (den) = [1 3 2]

En MATLAB:

matlab
% Definimos el numerador y denominador
num = [10];         % Numerador
den = [1 3 2];      % Denominador

% Creamos la función de transferencia
sys = tf(num, den);

% Graficamos polos y ceros
pzmap(sys)
grid on

3. Ejemplo con ceros y polos

Si quieres un sistema con ceros, tomemos:

G(s)=s+5s2+4s+20G(s) = \frac{s + 5}{s^2 + 4s + 20}G(s)=s2+4s+20s+5​

Aquí:

  • Numerador (num) = [1 5]

  • Denominador (den) = [1 4 20]

Código:

matlab
num = [1 5];       % (s + 5)
den = [1 4 20];    % (s^2 + 4s + 20)

sys = tf(num, den);
pzmap(sys)
grid on

4. ¿Para qué te sirve ver los ceros?

  • Polos dominan la dinámica, pero los ceros modifican la forma de la respuesta.

  • Un cero cercano al origen → puede acelerar la respuesta.

  • Un cero real positivo → puede causar sobreimpulso (respuesta no mínima fase).

  • Un cero puede incluso cancelar un polo si están en la misma posición.

Por eso, en control siempre se revisa la gráfica de pzmap:
👉 Si los polos están en el semiplano izquierdo → el sistema es estable.
👉 Los ceros ayudan a explicar comportamientos raros como oscilaciones, sobreimpulsos o anulaciones.
